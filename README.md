# UKL-3-November-Soal-1---Sedang-
Program yang ada di soal nomor 1 (sedang) ini berfungsi untuk menghitung hasil faktorial bilangan bulat positif suatu bilangan yang dimasukkan oleh pengguna
program yang saya buat ini menggunakan for untuk menghitung faktorial dan menggunakan tipe data long agar bisa membaca nilai yang lebih besar dibanding int
Cara Kerja Program ini adalah :
Pengguna diminta untuk memasukkan bilangan bulat yang ingin diinput melalui Scanner
Kemudian program ini menggunakan variabel bernama hasil untuk menyimpan hasil perkalian dari setiap angka dalam proses menghitung faktorial. 
Nilai awal dari hasil dibuat 1 supaya ketika dilakukan perkalian pertama kali, hasilnya tetap benar
Selanjutnya, program menjalankan perulangan menggunakan for mulai dari angka 1 hingga bilangan yang ingin dimasukkan pengguna.
Pada setiap langkah perulangan, variabel dikalikan dengan nilai i, yaitu angka yang sedang dihitung pada putaran tersebut.
Dengan begitu, nilai hasil akan terus bertambah besar mengikuti proses faktorial, misalnya untuk input 5 maka program akan menghitung 1 × 2 × 3 × 4 × 5 dan menghasilkan 120.
